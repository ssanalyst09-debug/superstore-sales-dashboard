# 📊 Superstore Sales Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard analyzing sales performance, profit trends, and customer behavior.

## 🛠 Tools Used
- Power BI

## 📊 Features
- KPI Cards (Sales, Profit, Orders)
- Interactive Slicers (Region, Category, Date)
- Multi-page dashboard (Overview, Product, Region, Customer, Insights)

## 🧠 Key Insights
- Technology category generates highest sales
- Furniture category has lower profitability
- West region performs best
- Some products generate high sales but low profit
